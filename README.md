# CountryFlags — Флаг страны по IP в трее Windows

Портативная утилита показывает флаг текущей страны (определяется по вашему внешнему IP) прямо в системном трее.  
Обновляется автоматически каждую минуту или вручную.

## Возможности
- Работает без установки (один .exe-файл)
- Все флаги уже внутри программы
- Автоматическое обновление каждые 60 секунд
- Ручное обновление через контекстное меню
- Автозагрузка при старте Windows (начиная с v1.1)
- Не требует Python и каких-либо зависимостей

## Как установить и запустить
1. Перейдите в [Releases](https://github.com/AleksandrSrk/CountryFlags/releases/latest)
2. Скачайте файл `CountryFlags.exe`
3. Запустите двойным кликом — иконка появится в трее
4. При первом запуске программа сама предложит (или автоматически добавит) себя в автозагрузку
