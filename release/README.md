# IP-Flag: Флаг страны по IP в трее

Портативная утилита для Windows — показывает флаг текущей страны (по IP) в системном трее. Обновляется автоматически каждую минуту или вручную через меню.

![Скриншот трея с флагом](https://via.placeholder.com/300x100?text=IP-Flag+in+Tray)  <!-- Замени на реальный скрин, если есть -->

## Как использовать
1. Скачай [IP-Flag.exe](https://github.com/AleksandrSrk/CountryFlags/releases/latest) (18 МБ).
2. Запусти двойным кликом — иконка появится в трее.
3. Правой кнопкой мыши: "Обновить сейчас" или "Выход".
4. Работает без установки! Требует интернет для проверки IP.

## Фичи
- Автообновление каждые 60 секунд.
- Поддержка 200+ стран (флаги в PNG/ICO внутри exe).
- Если нет интернета — остаётся последний флаг или красный индикатор.
